KETENTUAN UNTUK PRINT TIKET
1. Cek email anda, anda akan mendapatkan email dari pihak Tiketin.com
2. jika dalam beberapa saat belum diberi tiket, tunggu 5 menit, kami akan mengirimkan file dan code QR anda secepatnya
3. unduh file dan code QR tersebut otomatis tersimpan di perangkat anda
4. cara mendapatkan tiketnya yaitu, scan QR code nya ditempat print tiket
5. lalu setelah terprint kemudian di cek apakah tiketnya sesuai dengan file yang kami berikan
